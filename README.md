## TurboRepo for Mobi

### How to Start?

```sh
# If pnpm is not installed
npm install pnpm@8.9.0

# If this is the first time cloning the repo
pnpm install

# For local development
pnpm dev:<app name>

# For local build
pnpm build:<app build>
```
